# 一鍵移除 PDF 註釋

拖曳或選擇 PDF，立即取得乾淨版本。純前端處理，檔案不會上傳至任何伺服器。

## 功能

- 一鍵移除 PDF 內所有註釋（文字註解、螢光筆、手繪線條、連結等）
- 支援點擊上傳與拖曳上傳
- 純瀏覽器端處理，隱私安全
- 無需安裝，開啟即用

## 使用方式

1. 用瀏覽器開啟 `index.html`（雙擊或拖曳至瀏覽器視窗）
2. 或啟動本機伺服器後開啟：

   ```bash
   npx -y serve .
   ```
   然後在瀏覽器開啟顯示的網址（如 `http://localhost:3000`）

3. 選擇或拖曳 PDF 檔案至上傳區
4. 點擊「移除註釋並下載」
5. 下載的檔案會以「原檔名-無註釋.pdf」命名

## 技術

- [pdf-lib](https://pdf-lib.js.org/)：純 JavaScript PDF 編輯庫
- 無需後端，單一 HTML 檔案即可運作

## 注意事項

若 PDF 的註釋已被「壓平」（flatten）成頁面內容，則無法透過此工具移除。建議使用原始、未壓平的 PDF 檔案。
